# USAPopulationAPI

Aplikacja androidowa wykorzystująca API zawierające informacje o amerykańskich miastach.

Funkcją wykorzystaną w programie jest wypisanie aktualnej aplikacji dla podanego miasta.

Przykłądowe miasta:
- New York
- Alabama
- Texas
